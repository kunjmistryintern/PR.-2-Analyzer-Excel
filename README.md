## 📌 Overview

This project showcases an **interactive Excel dashboard** built to analyze sales data using advanced Excel features like Pivot Tables, Data Analysis Toolpak, and dynamic visualizations.

The dashboard helps in understanding **sales trends, customer performance, and regional insights**.

---

## 🎯 Objectives

* Analyze sales and profit data
* Identify high-value customers
* Track monthly sales trends
* Compare performance across regions and product categories
* Build an interactive dashboard for storytelling

---

## 🛠️ Tools & Features Used

* Microsoft Excel
* Pivot Tables
* Data Analysis Toolpak
* Conditional Formatting
* What-If Analysis
* INDEX + MATCH functions
* Charts (Bar, Line, Pie)
* Slicers for interactivity

---

## 📊 Dashboard Features

### 🔹 KPI Metrics

* Total Sales
* Total Profit
* Average Discount
* Total Quantity

### 🔹 Visualizations

* 📈 Monthly Sales Trend (Line Chart)
* 📊 Sales by Region (Bar Chart)
* 🥧 Sales by Product Category (Pie Chart)
* 📉 Sales by Product (Horizontal Bar Chart)
* 👥 Customer-wise Sales Analysis

### 🔹 Interactivity

* Month filter
* Product Category filter
* Region filter

---

## 📈 Key Insights

* West region generates the highest sales
* Sales peak observed in specific months
* Certain product categories contribute more to revenue
* High-value customers significantly impact total profit
